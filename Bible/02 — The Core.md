# The Solitaire Universe Bible

# 02 — The Core

Version 0.1 (Draft)

---

# Purpose

This chapter defines the Core.

The Core is not an object.

It is the origin from which every Solitaire Order emerges.

Everything within the Universe ultimately traces back to the Core.

---

# What is the Core?

The Core is the beginning of possibility.

It does not command the Universe.

It enables it.

Without the Core, the Five Orders cannot exist.

Without the Five Orders, the Universe has no balance.

---

# The Core is Reflection

The Core does not grow stronger because time passes.

It responds only to restoration.

Every completed Chapter strengthens an Order.

Every strengthened Order nourishes the Core.

The Core reflects the health of the entire Universe.

---

# The Core Speaks Without Words

The Core never explains itself.

Instead, it communicates through light.

Brightness.

Pulse.

Energy.

Motion.

The player should understand the state of the Universe simply by observing the Core.

---

# The Core Never Commands

The Core never asks the player to save it.

It never demands attention.

It quietly reflects the consequences of every action.

The responsibility belongs to the Keeper.

---

# Design Principles

The Core should always feel:

• Ancient.

• Calm.

• Timeless.

• Alive.

• Responsive.

It should never feel mechanical, aggressive, or decorative.

---

# Relationship to Gameplay

Gameplay restores the Orders.

The Orders strengthen the Core.

The Core reflects the state of the Universe.

The player witnesses this cycle after every meaningful achievement.

---

# One Sentence

The Core sustains the Five Orders.
