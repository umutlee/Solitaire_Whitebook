# The Solitaire Universe Bible

# 05 — The First Restoration

Version 0.1 (Draft)

---

# Purpose

This chapter defines Story Mode.

Story Mode is known within the Universe as the First Restoration.

---

# What is the First Restoration?

The First Restoration is the Keeper's first journey through the Five Orders.

Each Chapter has been carefully prepared.

Nothing is random.

The purpose is understanding,

not survival.

---

# Completion

Restoring every Chapter does not end the Universe.

It restores stability.

The Keeper has learned the Orders.

Balance has been restored.

---

# Beyond Restoration

When the First Restoration ends,

the Universe continues.

The Keeper continues.

The responsibility continues.

Random Seeds are not additional content.

They represent a living Universe whose future can no longer be predetermined.

---

# Design Principles

The First Restoration should feel:

Guided.

Meaningful.

Deliberate.

Hopeful.

It should never feel repetitive.

---

# Relationship to Endless Play

The First Restoration teaches.

The Living Universe challenges.

These are different stages of the same journey.

---

# One Sentence

The First Restoration is where the Keeper learns how to restore a living Universe.
